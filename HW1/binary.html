Binary Number System

The binary number system is a base-2 number system and it represents number using the combination of only two switches (on and off/ true and false) where On/True is denoted by the digit one (1) and Off/False is represented by the digit zero (0). Binary is preferred in computers due to computers naturally lending themselves to an On-Off system hence, it is easier to design circuits.

- Conversion from Binary to Decimal

1. Write down the binary number & list the powers of 2 above each number.
2. Write down that value below if there is a 1 and 0 if there is a 0.
3. Add up all the values to get the final answer.

- Conversion from Binary to Hexa-Decimal

Converting Binary: 0b 01101110
1. Group by 4. And convert each in decimal by listing powers of 2 and multiplying them with either 0 or 1, and then in Hexa-decimal.
2. 0110 -> 6(decimal) -> 6(hexadecimal) & 1110 -> 14(decimal) -> E(hexadecimal) 3. => 0x6E
