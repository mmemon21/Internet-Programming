Decimal Number System

The Decimal number system is a base-10 number system and it is used by humans only and not understood by computers. Any numerical quantity can be represented using these 10 digits (0-9).

Conversion from Decimal to Binary

1. Divide the decimal number by 2.
2. Write the result below it and the remainder on the right.
3. Keep dividing the result by 2 until the result is equal to 0.
4. The remainders from bottom to top will be the answer.

Conversion from Decimal to Hexa-Decimal

1. Divide the decimal value by 16.
2. Write the remainder in Hexadecimal.
3. Repeat process mentioned above in 1 and 2 until the result is 0.
