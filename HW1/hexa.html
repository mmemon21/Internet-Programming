Hexa-Decimal Number System

hexadecimal is a positional system that represents numbers using a base of 16. Unlike the common way of representing numbers with ten symbols, it uses sixteen distinct symbols, most often the symbols "0"–"9" to represent values zero to nine, and "A"–"F" to represent values ten to fifteen. Each Hexadecimal number can be represented in four bits in binary. Hence, it is a convenient way to express binary numbers in modern computers.
